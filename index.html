BACK-OFFICE

L'INFRASTRUCTURE INVISIBLE

Ce qui sépare un projet amateur d'une institution indestructible.

1. ANTI-CRISE

Bible de crise (réaction < 30s)
  • Symptôme : OBS plante.
    Décision : switch scène “Pause Tech” en 1 clic. Resp. : Régi. Message : “Pause technique, on revient”. Post-mortem : log crash.
  • Symptôme : Guest absent/retard.
    Décision : lancer Backup Content (réact prêt). Resp. : Prod. Message : “On commence avec un warm-up”.
  • Symptôme : Troll coordonné / raid.
    Décision : slowmode + ban wave. Resp. : Mod lead. Message : “Chat en mode lent, on nettoie”.
  • Symptôme : Copyright live.
    Décision : mute source, couper scène incriminée. Resp. : Tech. Message : “Audio temporairement coupé (copyright)”.
  • Symptôme : Buzz négatif post-live.
    Décision : statement court validé, pas d’impro. Resp. : Founder + Comms. Post-mortem : 24h après, plan correctif.
  • Symptôme : Chute brutale de viewers.
    Décision : inject mini-sequence chaude (jeu/sondage), pas de panique ni spam. Resp. : Host. Message : “On passe en mode interactif 2 minutes”.

2. TABLEAU DE BORD

  • KPIs Live (temps réel): CCV, rétention minute 5/15/30, chat velocity, taux drop après pub.
  • KPIs Post (J+1 / J+7 / J+30): CTR, watchtime, conversion Discord, récurrence des viewers.
  • KPIs Communautaires: Discord DAU/WAU, participation threads, taux retention onboarding.
  • KPIs Business: leads/opportunités, valeur perçue (mentions presse/partenaires).
  • Seuils d’action: <500 vues/clip = pivot format; >5% CTR = scale budget; drop rétention >20% = revoir hook.
  • KPIs à ignorer: vues brutes sans rétention, vanity likes, pic unique non récurrent.

3. TRANSMISSION

  • Onboarding 48h / rôle: fiche “À comprendre avant d’agir”, checklist poste, interdits absolus.
  • Versions Junior/Senior: tâches déléguables vs non déléguables clairement listées.
  • Playbooks: “Comment clipper Sherko”, “Moderation raid”, “Setup son/lumière”.
  • Remplaçabilité: binôme par poste + procédure de passation express.

4. ITÉRATION

  • Review hebdo (30min): ce qu’on garde / ce qu’on améliore / ce qu’on tue.
  • Mensuelle stratégique: positionnement, formats gagnants, allocation temps/budget.
  • Trimestrielle reset créatif: 1 nouveau format test, 1 innovation technique.
  • Kill switch: 4 essais sans traction → on coupe.

5. JURIDIQUE

  • Droits d’image: invités, mods, clips signés avant diffusion.
  • Ownership: live → shorts → archives, propriété Sherko.
  • Archives sensibles: stockage chiffré, accès restreint.
  • Lignes rouges éthiques: pas de buzz sale, pas de doxx, pas de fake drama.
  • Clause de sortie: séparation d’équipe / partenaires clarifiée.
